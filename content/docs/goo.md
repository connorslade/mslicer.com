(Now deprecated) Library for encoding and decoding Elegoo's `.goo` file format.

This crate has been deprecated in favor of my new [msla_format](/docs/msla-format) crate, which supports additional formats (.ctb and .nanodlp), and is auto generated from the mslicer source tree so it can be more easily updated.
This crate will no longer be updated.
